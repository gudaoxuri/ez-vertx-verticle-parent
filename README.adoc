= EZ Vertx-verticle Parent

=== EZ Vertx-verticle 构建框架，可使用Scala编写Vertx verticle程序基础包。

== 使用

[source]
----
<parent>
    <groupId>com.ecfront</groupId>
    <artifactId>ez-vertx-verticle-parent</artifactId>
    <version>0.2.0</version>
</parent>
----
